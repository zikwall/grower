## Grower

### Задумки

- Партиции хранить в директориях топиков
- Партиции разделить на сегменты (по весу или по количеству сообщений)
- Сегменты нумеровать так 0-10 сообщений 00000000000.growerlog и 10-20 сообщений 00000000010.growerlog
- Добавить индексные файлы к сегментам с офсетами и позициями байтов, формат сегмента:

```log
offset | position
1      | 0
2      | 45
```

- Для быстрого поиска и чтения использовать индексный файл с бинарным поиском, затем делать побайтовое смещение типа Seek
- При записи нужно сохранять смещение каждой записи
